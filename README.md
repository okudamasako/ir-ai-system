# AI投資情報配信システム

投資関連ニュースやIR情報をもとに、
AIが要約・整理・投稿支援を行うシステムの構成デモです。

## 内容

- 投資ニュース整理
- IR情報要約
- SNS投稿支援
- AIによる文章生成
- システム構成イメージ

## 使用技術

- HTML
- CSS
- JavaScript
- OpenAI API
- Vercel
